- Интерфейс: шрифт Onest Medium — мягкий и плотный, с поддержкой кириллицы

- Интерфейс: тонкий кастомный скроллбар вместо системного

- Инпут: раскладка под полем ввода `[Bypass] · · · [Sonnet 4.6] · [Low]`

- Модели: поддержка произвольного model ID с анимацией подтверждения
- Сессии: переименование через правый клик
- Настройки: боковая навигация вместо табов сверху
- Настройки: Claude CLI и Vael настройки разнесены по разделам
- Настройки: авто-обновление приложения, версия Vael в разделе обновлений
- Настройки: кастомный дропдаун вместо системного select
